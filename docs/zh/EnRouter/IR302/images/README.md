# Images convention (IR302)

Put images under language-specific folders:

- `images/zh/` for Chinese pages (`*.md`)
- `images/en/` for English pages (`*.en.md`) — you can use bilingual (CN+EN) images here

In Markdown, reference images by relative path, e.g.:

- Chinese page:
  - `![接线图](images/zh/wiring.png)`
- English page:
  - `![Wiring](images/en/wiring.png)`
